# Usage
<!-- usage -->
```sh-session
$ npm install -g compgenrc
$ compgenrc COMMAND
running command...
$ compgenrc (-v|--version|version)
compgenrc/0.1.1 darwin-x64 node-v15.12.0
$ compgenrc --help [COMMAND]
USAGE
  $ compgenrc COMMAND
...
```

## `compgenrc help [COMMAND]`

display help for compgenrc

```
USAGE
  $ compgenrc help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```
